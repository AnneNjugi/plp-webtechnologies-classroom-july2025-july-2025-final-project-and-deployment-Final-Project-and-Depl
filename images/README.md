## Note

Currently, the website uses CSS gradient placeholders. Replace these with actual images by:
1. Adding images to this folder
2. Updating the HTML to use `<img>` tags instead of placeholder divs
3. Example: `<img src="images/santorini.jpg" alt="Santorini Greece">`
